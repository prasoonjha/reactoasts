# reactoasts

> Customizable, beautiful Toasters for React.

[![NPM](https://img.shields.io/npm/v/reactoasts.svg)](https://www.npmjs.com/package/reactoasts) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save reactoasts
```

## Usage

```jsx
import React, { Component } from 'react'

import Toast from 'reactoasts'

class Example extends Component {
  render() {
    return <Toast 
        toastList={list}
        position={position}
      />
  }
}
```
